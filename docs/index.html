ブログ運営

私はブログを運営しております。実名を伏せるため、紹介は画像のみになりますが、Hugoを用いて開発・運営を行っております。Hugoは静的サイトジェネレータと呼ばれるサイト構成のための技術で、静的サイトはWordPressとは異なり、事前に記事を準備しておくことでサーバーの負担をかなり減らすことができます。またアクセス数が急激に増大した場合でもサーバーは安定して機能します。
